Mr. Skinner - The skinnable dropdown

mrSkinner is a reusable javascript plugin to create a nice-looking skinnable dropdown.

The dropdown can be applied on html select elements and has build-in skins such as retro, pink, zebra and more. Its easy to add your own skins with css.

Usage

Skins can be applied to all select elements with one method call.

$.mrSkinner({skin: "black"});

Skins can also been applied to a single or selected elements.

$("select#target").mrSkinner({skin: "rainbow"});

How does it work ?

The problem with the Html select element is that it has very limited ways to change the layout, apart from a changing the color, background color and font adjustments.

To provide more ways of playing with the layout mrSkinner creates a dropdown component using input box and a unordered list that allows full flexibility with css stylesheets.

The dropdown has additional css classess for hover, selected and zebra state.

Good Accessabilty: The dropdown supports keyboard shortcuts and tabindex to provice good accessability.

Graceful fallback: If javascript is disabled on the client the dropdown component gracefully degrades to a standard Html select element.
